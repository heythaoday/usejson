## Getting Started

### Prerequisites

- Node.js 16.x or later
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/heythaoday/usejson.git

# Navigate to project directory
cd usejson

# Install dependencies
yarn

# Start the development server
yarn dev
```
